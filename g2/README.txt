Usage:

1) Compile an executable from the g2 codebase by running:

$make

You should see the following output on your terminal:

g++ -Wall -O3 -std=c++11 -I tclap-1.2.1/include/ -o g2function g2function.cpp

2) You know have the executable g2function

3) Learn how to use it by typing:

$./g2function --help
